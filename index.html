  • To apply CSS to HTML, link the stylesheet using the `<“link“>` tag in the HTML `` section.
  • Errors in code are learning opportunities; debugging is a crucial developer skill.
  • The `rel` attribute in `<“link“>` defines the relationship type as a stylesheet.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init: Initialize a new Git repo.
  • git add: Stage changes for commit.
  • git add [file]: Adds a file to the staging area.
  • git commit: Saves the staged changes to the repository.
